# firebase.js

## Purpose
Initializes Firebase SDK and exports core Firebase services.

## Responsibilities
- Bootstraps Firebase app
- Provides Auth, Firestore, and Storage instances

## Firebase Backend Requirements
- Firebase project must exist
- Phone authentication enabled
- Firestore enabled
- Storage enabled

## Firestore Collections
None directly accessed here.

## Security Rules Expectation
Rules are applied at service level, not here.
